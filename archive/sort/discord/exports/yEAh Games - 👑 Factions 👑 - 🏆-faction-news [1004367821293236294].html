Factions: Representative application To apply for rep. of your faction, send a message with your application in #📋-rep-application. Once reviewed, @yEAh Events will send a message with the and emoji. If more members than before accept the application (for if there is already a representative), or if the majority of the faction does (when there is no rep.), they will become a representative. Please note that becoming a representative will not be instant. (edited)
